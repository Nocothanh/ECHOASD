# Echo AI — Build APK (No installs needed)

## How to get your APK in 10 minutes, completely free, no software required

---

### Step 1 — Upload to GitHub (free)

1. Go to **https://github.com** → Sign in (or create free account)
2. Click **"New repository"** (green button, top right)
3. Name it `echo-ai`, set to **Public**, click **Create repository**
4. On the next page click **"uploading an existing file"**
5. Drag and drop **all files from this ZIP** into the upload area
6. Click **"Commit changes"**

---

### Step 2 — Build APK on Expo (free)

1. Go to **https://expo.dev** → Sign in (or create free account)
2. Click **"Create a project"**
3. Choose **"Import from GitHub"**
4. Select your `echo-ai` repository
5. Click the **"Build"** tab → **"New build"**
6. Platform: **Android** → Profile: **Preview (APK)**
7. Click **"Build"** — takes ~5 minutes in the cloud

---

### Step 3 — Download & Install

1. When the build finishes, click **"Download build artifact"**
2. Copy the `.apk` file to your Android phone (via USB or cloud)
3. On your phone: open the file → tap **Install**
   - If asked: enable *"Install from unknown sources"* in Settings

---

## That's it! No npm, no Android Studio, no command line needed.

## First use

When you open Echo AI, tap ⚙️ and enter your API key:
- **Groq** (free): get key at https://console.groq.com
- **OpenAI**: get key at https://platform.openai.com
- **Anthropic**: get key at https://console.anthropic.com

Echo AI uses your key to call the AI directly — no middleman, no subscription.
